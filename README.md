# Expense Tracker

**Expense Tracker** is a simple web application built with React that allows users to track their expenses in a clear and intuitive way.

---

## Features

- Add new expenses (amount, category, description)  
- View a list of all expenses  
- Display total expenses  
- Categorize expenses  
- Responsive user interface  

---

## Technologies

- **Frontend:** React, JavaScript, CSS / SCSS  
- **Backend / Database:** Firebase (optional)  
- **Tools:** Create React App, Git
